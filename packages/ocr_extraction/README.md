## OCR Extraction

These environment vars need to be defined when using specific services:

Azure 
```
AZURE_OPENAI_KEY
AZURE_OPENAI_ENDPOINT
AZURE_OPENAI_MODEL
```

Bedrock Llama
```
BEDROCK_ACCESS_KEY
BEDROCK_SECRET_ACCESS_KEY
```

OpenAi
```
ASKEM_DOC_AI_API_KEY
```



To run the server:
```
python3 -m src.server
```
